# ai-cors

[![npm version](https://img.shields.io/npm/v/ai-cors.svg)](https://www.npmjs.com/package/ai-cors)
[![npm downloads](https://img.shields.io/npm/dm/ai-cors.svg)](https://www.npmjs.com/package/ai-cors)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)


Generate CORS configuration for your stack.

## Install

```bash
npm install -g ai-cors
```

## Usage

```bash
npx ai-cors "frontend on localhost:3000, API on localhost:8080"
npx ai-cors "React app on vercel, Express API on Railway"
```

## Setup

```bash
export OPENAI_API_KEY=sk-...
```
